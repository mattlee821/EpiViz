# Example data: Simulated epidemiology results for track 2

A simulated dataset of effect estimates and confidence intervals for 120
variables.

## Usage

``` r
EpiViz_data2
```

## Format

A data frame with 120 rows and 8 variables.

- label:

  Variable names

- class:

  Greater category/grouping

- subgroup:

  Minor category/grouping

- beta:

  Effect estimate

- se:

  Standard error

- pvalue:

  P-value

- lower_confidence_interval:

  Lower 95% confidence interval bound

- upper_confidence_interval:

  Upper 95% confidence interval bound
